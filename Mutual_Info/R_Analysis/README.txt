These are the R codes used to do the group analysis for differences in SWP where SWP is calculated using the Mutual
Information values of each pair of nodes.

barcharts.R creates bar graphs of the data with error bars
groupFtests.R does the F-test for differences between groups with permutation tests to confirm
groupttests.R does the t-tests for differences between groups for individual tasks
RepMeasPermTest.R does the Repeated Measures ANOVA for group differences
